### Installation

_How to setup this project_

1. Clone the repo
   ```sh
   git clone https://github.com/bishakhne0gi/Linkhub.git
   ```
3. Install all the NPM packages
   ```sh
   npm install
   ```
  ```
4. Run
  ```
  npm start 
   ```
5. Download ExpoGo in your mobile. Scan the QR
