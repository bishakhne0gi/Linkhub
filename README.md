### Installation

_How to setup this project_

1. Clone the repo
   ```sh
   git clone https://github.com/bishakhne0gi/Linkhub.git
   ```
2. Install all the NPM packages
   ```sh
   npm install
   ```
  ```
3. Run
  ```sh
  npm start 
   ```
4. Download ExpoGo in your mobile. Scan the QR
